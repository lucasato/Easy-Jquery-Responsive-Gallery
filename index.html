Easy Jquery Photo Gallery Plugin - Responsive Gallery

Call example:

 $(document).ready(function()
  {
   $('.pluginSelector').easyGalleryJquery(
    {
      title:'My first gallery',
      preloader:'images/loader.gif', //optional
      percent: 80 //optional, default value: 80% of screen
     }
   );
 });
Html:

<section class="pluginSelector">
 <a href="images/0.jpg">
  <img class="img-responsive" src="images/mini_0.jpg" alt="My Photo Caption at Window Bottom #1"/>
 </a>
 <a href="images/1.jpg">
  <img class="img-responsive" src="images/mini_1.jpg" alt="My Photo Caption at Window Bottom #2"/>
 </a>
</section>
Include CSS And JS Files (Works with jQuery 1.3+):

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.2.1/jquery.min.js">
<link rel="stylesheet" href="css/easyjquerygallery.min.css">
<script src="js/easyjquerygallery.min.js">
